Mobility of Compute

What is Mobility of Compute?

Mobility of Compute is the encapsulation of a system environment in such a manner to make it portable between systems. The operating system environment can contain the necessary applications for a particular workflow, development tools, and/or raw data. Once this environment has been developed it can be easily copied and run from any other Linux system.
